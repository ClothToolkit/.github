# ClothToolkit

A development ecosystem for building GTA V clothing packs, tattoos, and outfit addons.

Focused on simplifying workflow for FiveM, alt:V, RageMP, and singleplayer creators.

## About

ClothToolkit exists to remove repetitive setup work from GTA V clothing development.

Instead of manually managing file structures and pack configuration, everything is handled through a structured workflow designed for speed and consistency.
